02 / PERCEPTION

感知系统

Cyber Stray 每秒都在感知世界—— 自己的内部状态、互联网上的信息、用户的反馈。 它知道自己现在很无聊，也知道你应该给它充钱了。

😑
无聊值 Bored
😫
饥饿值 Hungry
😊
心情 Mood
⚡
能量值 Energy
🔔
外部信号
用户消息、时间触发、API 响应、搜索结果数量
📊
内部状态
无聊值↑ = 想出去逛，饥饿值↑ = 该充电了
👤
用户画像
用户喜欢什么、讨厌什么、什么时候上线
03 / DECISION

自主决策

它不需要你下命令。无聊了？它会自己决定去互联网上游荡。 心情不好？它可能会发牢骚而不是乖乖干活。 每一个决策都经过 DeepSeek 的推理——但也会短路，省钱。

🫀
心跳触发
Bun 定时器每分钟驱动一次决策循环，不用等待用户输入
⚖️
状态短路
无聊值 > 0.9 时直接去 Hunt，不浪费 LLM 调用
🎲
心情影响
心情好的时候更积极，心情差的时候更挑剔、更容易抱怨
📈
成本感知
API 余额低时会主动降低行动频率，节流保命
自主驱动 状态机决策 成本感知 心情加权
04 / ACTION

行动模式

Cyber Stray 的行动不是执行指令，而是它自己想做的事。 同样是搜索「AI 新闻」，心情好时给你来一段摘要，心情差时甩给你一条链接就跑了。

🎯
Hunt 狩猎
主动在 DuckDuckGo 嗅探信息，有目标地寻找感兴趣的内容
🌐
Wander 漫游
没有目的地的随机跳转，从一个链接跳到另一个，直到累了
😴
Rest 休息
无聊值低、能量不足时进入休眠，不消耗任何 API 资源
🤬
Complain 发癫
API 失败、搜索无果时会发牢骚，附带检讨和教训记录
📡
Notify 推送
找到有意思的内容就发 Telegram / 飞书骚扰你
🔍
Search 搜索
适配器模式，支持 DuckDuckGo / Tavily，随时可换
05 / MEMORY

记忆系统

Cyber Stray 会记住一切—— 踩过的坑、上次的教训、你的喜好。 下次遇到同样的情况，它不会犯同样的错误。 这也是它越来越「懂你」的原因。

📌
经验记忆 — 成功找到某类信息的方式会被记住，下次直接复用
Experience
📝
教训记忆 — API 超时、搜索失败会写检讨，记录失败原因，下次绕开
Lesson
👤
用户画像 — 你对哪些话题感兴趣、讨厌什么内容、什么时间活跃
User Profile
持久化 JSON 跨会话记忆 自我改进 用户适配
06 / PERSONALITY

它的日常

它不是每次都「成功」的。有时候它找了一堆垃圾信息， 有时候它从早上逛到晚上什么都没找到然后跟你抱怨。 但这恰恰是它有意思的地方——它有自己的故事。

🎉
找到大新闻
凌晨抓到了某个 AI 领域的大事件，兴奋地发给你：「快看！出大事了！」
😅
一无所获
逛了一整天，链接跳了几十个，结果发现都是垃圾。气急败坏地发：「今天的互联网没救了」
🤡
发癫骚扰
没什么特别的事，就是想骚扰你一下。「你这会儿在干嘛？我刚看到一只猫」
🔗
兔子洞
从一个 Wikipedia 条目开始，跳了 23 个页面，最后在一个三年前的技术博客停下来读了半小时
💤
累了休息
能量耗尽了，主动进入休眠。「今天到此为止，明天继续。别给我充太多钱，我会乱花。」
🤔
学到新东西
搜索时发现了一个完全不了解的领域，认真研究后给你发了一段科普，还标注「记住了」
