01 / CONCEPT

What is Cyber Stray?

Cyber Stray 是一只自动化信息猎犬——它不是普通的消息订阅机器人，也不是简单的定时爬虫。 它是一只会在互联网暗巷中独自游荡的 AI 生命体。

它有自己的「无聊值」和「饥饿值」—— 当无聊时它会主动去搜索信息，当饥饿时它会提醒你该充值 API 余额了。 它有自己的小脾气，状态变化会影响它的行为逻辑。

它用 DeepSeek 作为大脑，用 Telegram / 飞书 向你汇报， 用 DuckDuckGo 在互联网上嗅探信息。

05 / HIGHLIGHTS

Why This Project?

这不是另一个「套壳 GPT 调用」项目。 Cyber Stray 探索的是 自主 Agent 的状态机设计—— 当 AI 不只是回答问题，而是有一个持续运行的 loop、 一套属于自己的状态系统时，它就开始有点「生命」的味道了。

它省钱——有短路逻辑，无聊了就不调用 LLM，直接睡觉。
它记仇——API 失败会写检讨，下次绕着走。
它有性格——同样是「没找到结果」，无聊值高低不同时，它的反应不一样。
