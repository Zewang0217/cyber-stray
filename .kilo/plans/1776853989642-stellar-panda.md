# 项目了解计划

## 项目概览

**cyber-stray（赛博街溜子）** 是一个自主 AI Agent，像一只在互联网上游荡的"信息猎犬"，自动发现有趣内容并推送给用户，同时具有自己的情绪、行动逻辑和个性。

---

## 技术栈

| 层次 | 技术 |
|------|------|
| 运行时 | Bun |
| 语言 | TypeScript（严格模式） |
| LLM 集成 | Vercel AI SDK（`ai`, `@ai-sdk/openai`）→ DeepSeek |
| Schema 验证 | Zod |
| 日志 | consola |
| 搜索 API | Tavily（规划中） |
| 推送渠道 | 飞书 Webhook、Telegram Bot（规划中） |
| 状态存储 | JSON 文件 + Markdown 快照 |
| 测试 | bun test |

## 核心数据模型

- `AgentState`：boredom (0–100)、energy (0–100)、mood、temper、stubbornness、记忆数组、统计、时间戳
- `Mood`：`curious | grumpy | playful | lazy | excited | emo`
- `ActionType`：`hunt | rest | complain | celebrate | ignore | procrastinate`
- `Decision`：LLM 规划器输出（动作 + 参数）
